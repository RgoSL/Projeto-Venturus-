### Como Rodar o Projeto ⚙️🔧

```bash

# Requisitos
É necessário ter o Node.Js e o React.Js instalados na máquina

# Clone o repositório
git clone https://github.com/RgoSL/Projeto-Venturus-.git

# Instale as dependências
npm install

# Rode o projeto
npm start

```
